# Reward — 奖励池前端

## Pages
| 文件/组件 | 职责 |
|-----------|------|
| `RewardPoolView.vue` | 主页面（BalanceCard + RewardStats + Gift概览 + RewardTimeline） |
| `GiftListView.vue` | 礼物清单独立页 |

## Components
| 文件 | 职责 |
|------|------|
| `BalanceCard.vue` | 3张统计卡（余额/获得/提取） |
| `RewardStats.vue` | 来源分布（里程碑/小确幸） |
| `RewardTimeline.vue` | 流水时间线 |
| `GiftCard.vue` | 礼物卡片（分类图标+进度条+状态标签） |
| `GiftForm.vue` | 创建/编辑弹窗 |
| `GiftExchangeModal.vue` | 兑换确认弹窗 |
