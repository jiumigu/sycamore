# Inbox — 收件箱前端

## Views
| 文件 | 职责 |
|------|------|
| `InboxDashboard.vue` | 收件箱主页（快速录入 + 统计卡 + 筛选 + 列表 + 弹窗 + 完成备注/小确幸/奖励联动弹窗） |

## Components
| 文件 | 职责 |
|------|------|
| `QuickInput.vue` | 快速录入（3 秒内完成，支持类别/优先级/截止日期） |
| `InboxStats.vue` | 4 个统计卡片（待处理/本周完成/逾期/总条数） |
| `InboxFilter.vue` | 搜索 + 状态（待处理/已完成/已处理/全部）/类别/优先级筛选 + 整理模式切换 |
| `InboxItem.vue` | 单条记录（含优先级色/逾期标记/操作菜单/完成备注绿色显示） |
| `InboxList.vue` | 列表容器（加载骨架屏/空状态/选中逻辑） |
| `BatchActions.vue` | 批量操作栏（完成/归档/删除，带滑入动画） |
| `ConvertModal.vue` | 转为目标/里程碑/能量 弹窗（里程碑模式含目标选择器 + 目标日期） |
| `InboxForm.vue` | 编辑/详情弹窗（含处理历史展示） |
